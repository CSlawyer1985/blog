文科生面对那个黑窗口：门槛比你想象的低得多

!封面

有件事我差不多每次分享都会遇到。

讲完之后有人上来问："陈律师，你讲的东西我都很认同。但你是有编程基础的吧？"

我说没有。大学学的是法律，数学一般，没写过一行代码。

对方通常不信。"那你怎么会用终端？怎么看得懂命令行？"

这个问题背后的预设很有意思——好像"会打开终端"是一个需要特殊身份才能做的事。

那个黑窗口吓退了太多人

我完全理解一个文科生看到终端窗口为什么本能地想关掉。

黑底白字、光标在闪、没有任何按钮和图标、你完全不知道应该输入什么。它看起来就像"这不是给你用的"。

我第一次打开终端，盯着那个光标大概十秒钟，默默关掉了。心想这个东西应该是程序员用的，我不碰应该也没事。

但后来我碰了。因为用 Agent 类产品——不管是 Claude Code、Codex 还是 Trae——你绕不开。Agent 就是在终端里跑的。你不进这个门，就一直待在 Chatbot 的世界里：每次打开都是重新开始，每次关掉什么都没留下。

硬着头皮打开之后，我发现了一件事。

你不需要会命令行。你只需要会打字。

你打开那个窗口那一刻，就已经会了

Agent 类工具的交互方式是什么？你说话，它干活。

你打开终端，输入"帮我分析一下这个 PDF 里的合同条款"，它自己读文件、拆结构、找风险点、给你出一份带批注的修订版。

在这个过程中，你输入的是中文。你不需要输入任何"命令"。不需要知道什么叫 cd、ls、grep。你只需要把需求说清楚，像跟助理交代工作一样。

Agent 和 Chatbot 的区别不是"一个在网页上一个在终端里"。区别是：Agent 能记住你的习惯、能控制你的文件、能在你的工作目录里直接操作。终端只是它的栖身之地，不是你学习的目标。

意识到这一点之后，那个黑窗口突然不一样了。它不再是一个"我不配用"的东西。就是一个空白画布——你在上面用中文描述要什么，它帮你实现。

正向激励来得极快

四明山法师 AI 夜校里有一个我印象最深的例子。

一个做知产的女律师，第一天上课的时候连"下载一个软件"都不太顺。她对电脑的全部认知就是 Office 和微信。第一节课讲 Agent 的基本概念，她的表情就是"你们在说什么"。

布置了第一次作业：用 Agent 搭一个最简单的法律咨询工作区。

她交了。做出来的东西很简陋，但能用。我问她怎么做的，她说碰到不懂的就截图问 AI，AI 告诉她怎么做就怎么做。

第二次作业是按自己需求改造这个工作区。她改成专门做商标侵权分析的——把知识库里商标法的相关书籍加进去，设计了初步判断的流程。

交完第二次作业那天晚上，她在群里发了一段话。大意是："我以前觉得这些是给技术背景的人准备的。现在我女儿问我为什么妈妈也在写代码，我说这不是写代码，这是在工作。"

你作为一个文科生，突然能写一个网页了，突然能自动处理一百份合同了，突然能做一张以前得找美工才能做的海报了——那个爽感来得极快。它不是"我努力学习考了好成绩"的延迟满足，是"我说了一句话它就做好了"的即时反馈。

这个时代最好的事情：你旁边就坐着一个全科老师。

学 AI 的方法就是用 AI

你打开 Agent，碰到第一个不会的东西——比如不知道 Skill 是什么——直接在对话框里问它。

它会告诉你。讲得比任何教程都清楚。

照着做，做错了它会说哪里错了。你再改，它再跑。来回三五次，通了。

然后你就学会了。不是"听了一堂课回去练习"那种学会，是"动手做出来了肌肉记住了"那种学会。

我自己就是这么学的。过去一年半，没有上过任何一门系统的编程课，没看过一本技术书。想问什么直接问 AI。碰到报错截个图扔给它。它告诉我哪里有问题，我改，再跑。

慢慢就培养出一种手感——知道这件事它能做、那件它做不了、这个方向大概能走通、那个坑不用踩。这个手感不是学出来的，是碰出来的。

大部分人说"我技术不行"，其实是"我还没开始"

那些说"我技术不行所以学不了 AI"的人，通常还没有真正打开过那个窗口。

恐惧不是因为难，是因为陌生。一个你从没用过的界面，看起来像三十年前的操作系统，任何人都会有本能的抗拒。但这种陌生感消退得极快——大概两个小时的工夫。

两小时后你会发现，它其实就是在跟你聊天。只不过聊天的背景从白色变成了黑色。

我没办法让你不觉得这个窗口陌生。但我可以告诉你一个数据：四明山法师 AI 夜校两期二十多个学员，百分之百完成了两次作业。其中大部分是文科生、律师、法务。没有一个人说"太技术了我学不了"。

不是他们天赋异禀。是这东西本来就没有技术门槛。

门槛在心理上，不在能力上。

你唯一需要的是一个理由

如果你目前用 AI 的方式还是打开 Kimi 或者豆包的网页，问一个问题，关掉，下次打开重新来——那你确实不需要终端。Chatbot 够用。

但如果你开始觉得"每次重新来"这个状态不太对。如果你感觉用了很久 AI 但没有留下任何属于你的积累。如果你想让 AI 像带助理一样——每次都在变得更好用、更懂你、产出更接近你要的东西——

那你就需要一个 Agent。而 Agent 需要你打开那个黑窗口。

这件事的上手难度，真的就是"在文本框里打字"的难度。剩下的交给 AI。

写到这儿想起一件事。我一个做了二十几年诉讼的老同事，半年前连 ChatGPT 都没用过。上个月跟我说："我觉得我这个助理越来越聪明了，比我带的有些实习生还好使。"

他说的"助理"不是真人。是他养了三个月的 Agent。

这就是我想讲的核心：那个黑窗口吓退了太多优秀的法律人。而它根本不应该是一个门槛。
